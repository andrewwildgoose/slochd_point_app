# slochd_point_app

Blank SvelteKit starter for the Slochd Point calculation web application.

## Prerequisites

- Node.js 20 or newer
- npm 10 or newer

## Getting started

1. Install dependencies:

   ```sh
   npm install
   ```

2. Review the default environment values in `.env` and update them if needed for your machine.

3. Start the development server:

   ```sh
   npm run dev
   ```

4. Open the local URL printed in the terminal (usually `http://localhost:5173`).

## Useful commands

- `npm run dev` — start the local development server
- `npm run build` — create a production build
- `npm run preview` — preview the production build locally
- `npm run check` — run SvelteKit and TypeScript checks
